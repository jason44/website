Image Noise

The problem with approximating finite differentials is that noise in the image would result in an undesirable results for the derivative magnitude graph. ie: ![[Pasted image 20230420114715.png|300]]

The solution is to convolve the image with a smoothing kernel (usually a gaussian distribution). In order to achieve this, we let the kernel be a [[differentiable function]] and sample some points. This allows us to reduce the smoothing to a single convolution mainly, This is only possible because differentiation is convolution and convolution is associative
