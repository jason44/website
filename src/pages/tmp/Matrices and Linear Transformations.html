Matrices are used to represent linear transformations. Since every element in a vector space can be represented as a linear transformation of a set of basis vectors, we let the columns of the matrix equal the coordinate of our chosen basis vectors. When we do matrix multiplication on a vector, we basically create a system of linear equations that represents the transformed vector.
