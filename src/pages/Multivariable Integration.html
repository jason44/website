Regions

For a double integral, we can take a non-rectangular region of the integrand by specifying the bounds of the inner integral with curves For triple integrals, we can specify the bounds of a region with a surface The main idea is that for each inner integral, we will use variables from the outer integrals to specify the bounds.

If the inner integral of a double integral is bounded by points, then the result is a rectangular region. If the inner integrals of a triple integral is bounded by points, then the result is a box region.
